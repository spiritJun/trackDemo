## 开发
```bash

    # 安装yarn 
    npm install -g yarn

    # 安装依赖
    yarn install

    # 本地开发 开启服务
    npm run serve 或 yarn serve
```
浏览器访问 http://localhost:8080

## 发布
```bash

    # 构建测试环境
    npm run build:stage 或 yarn build:stage

    # 构建生成环境
    npm run build:prod 或 yarn build:prod
```

## 预览
```
    # 本地预览打包文件
    npm run preview 或 yarn preview
```
